## 3. ภาพรวมระบบ
ระบบจัดการงานและผู้ใช้ (Task and User Management System) เป็นระบบที่ออกแบบมาเพื่อช่วยในการจัดการงานและผู้ใช้ในองค์กร โดยมีฟีเจอร์หลักดังนี้:
- การจัดการผู้ใช้ (User Management): ระบบสามารถสร้าง แก้ไข และลบผู้ใช้ได้อย่างง่ายดาย รวมถึงการกำหนดสิทธิ์การเข้าถึงต่าง ๆ
- การจัดการงาน (Task Management): ระบบสามารถสร้าง แก้ไข และลบงานได้อย่างง่ายดาย รวมถึงการกำหนดสถานะของงานและการมอบหมายงานให้กับผู้ใช้
- การตรวจสอบและบันทึกกิจกรรม (Activity Log): ระบบสามารถบันทึก
กิจกรรมต่าง ๆ ที่เกิดขึ้นในระบบ เช่น การสร้างงาน การแก้ไขงาน และการลบงาน เพื่อให้สามารถตรวจสอบย้อนหลังได้
- การยืนยันตัวตนด้วย JWT (JWT Authentication): ระบบใช้ JSON Web Tokens (JWT) ในการยืนยันตัวตนของผู้ใช้ เพื่อให้การเข้าถึงข้อมูลและฟีเจอร์ต่าง ๆ 
เป็นไปอย่างปลอดภัย
